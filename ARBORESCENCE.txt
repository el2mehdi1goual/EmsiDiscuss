## 🎯 Modèles et Relations

```
User (Django Auth)
  │
  ├─→ UserProfile (1-to-1)
  │
  ├─→ Topic (Author: Foreign Key)
  │   ├─→ Category
  │   └─→ SubCategory
  │
  ├─→ Reply (Author: Foreign Key)
  │   └─→ Vote (Multiple Votes)
  │
  ├─→ Report (Reporter: Foreign Key)
  │   └─→ Vote / Topic / Reply
  │
  ├─→ Ban (User: Foreign Key)
  │   └─→ Banned_by (Admin)
  │
  └─→ Notification (User: Foreign Key)
      └─→ Type: reply, topic, mention, message
```
